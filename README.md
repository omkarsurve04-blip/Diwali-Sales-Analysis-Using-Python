# Diwali-Sales-Analysis-Using-Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a Diwali Sales dataset to uncover customer purchasing behavior, sales trends, and product performance during the festive season.

The goal is to help businesses make data-driven marketing, inventory, and targeting decisions to maximize revenue.

## Objectives

- Understand customer demographics and buying patterns
- Identify high-revenue customer segments
- Analyze top-performing product categories
- Discover trends based on age, gender, occupation, and city tier
- Generate actionable business insights

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features

- The dataset includes:
- User ID
- Gender
- Age Group
- Marital Status
- State & City Category
- Occupation
- Product Category
- -Orders
- Amount (Sales Value)

## Key Analysis Performed

- Data cleaning & preprocessing
- Missing value handling
- Feature exploration
- Groupby aggregations
- Sales trend analysis
- Demographic segmentation
- Visualization of insights using charts & graphs
